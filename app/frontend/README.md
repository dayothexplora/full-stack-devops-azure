# Frontend

Simple web interface for the Full Stack DevOps Platform.

## Features

- Displays platform title
- Button to fetch data from backend API
- Shows API response in real-time

## Local Development

### Prerequisites
- A web browser
- Backend API running on `http://localhost:5000`

### Setup and Run

1. Start the backend API first (see `/app/backend/README.md`)

2. Open `index.html` directly in your browser:
   - Double-click the file, OR
   - Right-click → Open with → Your browser

3. Click the "Fetch Data" button to test the connection to the API

## Files

- `index.html` - Main web interface with JavaScript to call the API
